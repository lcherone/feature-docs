# Products

Products is an admin feature for reviewing, creating, and editing the settings shown in the screens below.

![Products overview](pages/001-cp-products-admin-27f8eb6b/images/page-desktop.png)

*Products overview*

## What This Feature Does

- Use the listing to search, filter, and review existing Product entries.
- Select Create new to add a Product entry, then complete the labelled settings and save.
- Open an existing Product entry to review or update its settings.

## Key Settings

- **Title:** Updates Title.
- **Category (optional):** Updates Category.
- **Url Name (optional):** Updates Url Name (optional).
- **Status:** Updates Status.
- **Is Module?:** Updates Is Module?.
- **Restriction UK (optional):** Updates Restriction UK (optional).
- **Restriction EU (optional):** Updates Restriction EU (optional).
- **Restriction US (optional):** Updates Restriction US (optional).
- **Message UK (optional):** Updates Message UK (optional).
- **Message EU (optional):** Updates Message EU (optional).
- **Message US (optional):** Updates Message US (optional).
- **Variant 1 (optional):** Updates Variant 1 (optional).
