# Products

[Products overview](../../index.md) / Create Product

URL: [https://sohohome.com/cp/products-admin/edit/new](https://sohohome.com/cp/products-admin/edit/new)

Use this page to manage Products.

![Products overview](images/page-desktop.png)

*Products page overview*

## Using This Page

1. Open a Product entry from the listing, or select Create new.
2. Complete the labelled settings for the entry.
3. Select Save to apply the changes.

## What You Can Do

### Create a new entry

Select Create new to add a Product entry, then complete the labelled settings and save.

### Edit an existing entry

Open an existing Product entry to review or update its settings.

- Save applies the changes.
