# Soho Home CP Documentation

A practical guide to the Soho Home CP features captured so far.

## What Each Page Includes

- A direct URL for the live CP screen.
- A redacted page screenshot for orientation.
- A short explanation of what the feature is for.
- Main workflows, important fields, and example listing rows where they help explain the feature.

## Features Covered

1. **[Access Tokens](pages/001-cp-access-tokens-c852f881/README.md)**: Access Tokens store integration access and refresh tokens for services that need authenticated API calls.
   URL: [https://sohohome.com/cp/access-tokens](https://sohohome.com/cp/access-tokens)
2. **[Accounting Logs (Sage & Avalara)](pages/003-cp-accounting-logs-140ba15d/README.md)**: Accounting Logs (Sage & Avalara) record Sage and Avalara accounting activity so integration issues and resolution status can be reviewed.
   URL: [https://sohohome.com/cp/accounting-logs](https://sohohome.com/cp/accounting-logs)
3. **[Adjustments](pages/005-cp-adjustments-summary-admin-6754dd0f/README.md)**: Adjustments summarise order adjustments and related finance-system transfer activity for review.
   URL: [https://sohohome.com/cp/adjustments-summary-admin](https://sohohome.com/cp/adjustments-summary-admin)
4. **[Outbound API Logs](pages/007-cp-ais-client-outbound-logs-458c975b/README.md)**: Outbound API Logs record outbound AIS API requests so sent data, failures, and debug activity can be reviewed.
   URL: [https://sohohome.com/cp/ais-client-outbound-logs](https://sohohome.com/cp/ais-client-outbound-logs)
5. **[Product API Logs](pages/009-cp-ais-client-product-logs-c8cce05a/README.md)**: Product API Logs record product-related AIS API activity so product sync issues can be investigated.
   URL: [https://sohohome.com/cp/ais-client-product-logs](https://sohohome.com/cp/ais-client-product-logs)
6. **[Client API Settings](pages/010-cp-ais-client-settings-1989fca0/README.md)**: Client API Settings control AIS client sync behaviour, including Business Central sync and scheduled pricing update settings.
   URL: [https://sohohome.com/cp/ais-client-settings](https://sohohome.com/cp/ais-client-settings)
