# Soho Home CP Documentation

A practical guide to the Soho Home CP features captured so far.

## What Each Page Includes

- A direct URL for the live CP screen.
- A redacted page screenshot for orientation.
- A short explanation of what the feature is for.
- Main workflows, important fields, and example listing rows where they help explain the feature.

## Features Covered

1. **[Customers](pages/001-cp-customers-cbb2a4e5/README.md)**: Customers is used to find customer accounts and review the account, address, membership, basket, and order details available to admins.
   URL: [https://sohohome.com/cp/customers](https://sohohome.com/cp/customers)
2. **[Image Asset Admin](pages/002-cp-image-asset-admin-1c8e0118/README.md)**: Image Asset Admin manages reusable image assets used across content and merchandising areas.
   URL: [https://sohohome.com/cp/image-asset-admin](https://sohohome.com/cp/image-asset-admin)
3. **[Returns](pages/003-cp-returns-admin-799f6596/README.md)**: Returns is used to review return records and follow their processing status.
   URL: [https://sohohome.com/cp/returns-admin](https://sohohome.com/cp/returns-admin)
4. **[Swatches](pages/004-cp-swatches-716721cf/README.md)**: Swatches manages the swatch records used for fabric, finish, and material sample journeys.
   URL: [https://sohohome.com/cp/swatches](https://sohohome.com/cp/swatches)
5. **[Redirects](pages/005-cp-redirects-7f08f116/README.md)**: Redirects manages URL redirects so old or changed links send visitors to the correct page.
   URL: [https://sohohome.com/cp/redirects](https://sohohome.com/cp/redirects)
