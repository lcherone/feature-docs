## Key Settings

### Edit Redirect

#### From

![From](images/field-001-from.png)

*From setting*

Add the from.

**Validation:** Required.

#### To (optional)

![To optional](images/field-002-to-optional.png)

*To (optional) setting*

Add the to (optional).

**Notes:** optional

#### Method

![Method](images/field-003-method.png)

*Method setting*

Choose the option that matches this method.

**Options:** Exact Match, Head Match

#### Strip query parameters

![Strip query parameters](images/field-004-strip-query-parameters.png)

*Strip query parameters setting*

Turn this on when strip query parameters should apply. Leave it off when it should not.

#### Permanent

![Permanent](images/field-005-permanent.png)

*Permanent setting*

Turn this on when permanent should apply. Leave it off when it should not.
