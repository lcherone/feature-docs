## How It Works

- The key fields are Email, Country, Date, Source, and How did you hear about us?, which explain what the record is for and how it can be used.

## Using This Page

1. Open Newsletter Signups from the CP navigation.
2. Scan the fields in the table to find the newsletter signup you need.

## What You Can Do

### Review newsletter signups

Review the visible fields to check what already exists.

- Field: Email
- Field: Country
- Field: Date
- Field: Source
- Field: How did you hear about us?
