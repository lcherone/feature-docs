## Using This Page

1. Open Pending Shipments from the CP navigation.
2. Search or filter until you find the pending shipment you need.

## What You Can Do

### Review pending shipments

Search or filter the visible fields to find the pending shipment you need.

- Field: Shipment Reference
- Field: Order Reference
- Field: Order Status
- Field: Date
