## What You Can Do

### Review navigation

Review the visible fields to check what already exists.

- Field: Targeted Content
- Field: UK
- Field: EU
- Field: US
- Field: Title
- Field: URL
- Field: Mobile Link Label
- Field: Created
- Field: Updated
