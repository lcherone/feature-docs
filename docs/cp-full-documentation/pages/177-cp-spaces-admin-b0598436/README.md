## How It Works

- The key fields are Title, Intro, Url name, Rooms, and Listing Image, which explain what the record is for and how it can be used.

## Using This Page

1. Open Spaces from the CP navigation.
2. Search or filter until you find the space you need.

## What You Can Do

### Review spaces

Search or filter the visible fields to find the space you need.

- Field: UK
- Field: EU
- Field: US
- Field: Status
- Field: Title
- Field: Product Image Order
- Field: Visibility
- Field: Colour When Transparent
