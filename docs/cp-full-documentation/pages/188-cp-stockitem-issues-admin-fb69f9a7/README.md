1. Open SKUs with Issues from the CP navigation.
2. Scan the fields in the table to find the skus with issue you need.

## What You Can Do

- Field: SKU Status
- Field: SKU Desc. Character Count
- Field: Group Desc. Character Count
- Field: UK
- Field: EU
- Field: US
- Field: Created
- Field: Updated
- Field: First Active
