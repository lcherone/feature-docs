## Using This Page

1. Open Orders from the CP navigation.
2. Search or filter until you find the order you need.
3. Open a row when you need to check the full details.

## What You Can Do

### Review orders

Search or filter the visible fields to find the order you need, then open the row to check the full details.

- Field: Name
- Field: SKU
- Field: Type
- Field: Status
- Field: QTY
- Field: Total

### Review an existing order

Open an existing order when you need to check the full details.
