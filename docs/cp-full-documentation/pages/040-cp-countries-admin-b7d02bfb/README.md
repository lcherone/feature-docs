## Using This Page

1. Open Countries from the CP navigation.
2. Search or filter until you find the country you need.

## What You Can Do

### Review countries

Search or filter the visible fields to find the country you need.

- Field: Label
- Field: Status
- Field: Code
- Field: Locale
- Field: Currency
- Field: Use Tax Inclusive Pricing?
- Field: Tax Rate
- Field: Global-e?
