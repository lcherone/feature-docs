## How It Works

- The key fields are Title, Intro, Url name, Rooms, and Listing Image, which explain what the record is for and how it can be used.

## Using This Page

1. Open the existing space you need to change.
2. Work through the fields that are relevant to the change.
3. Save once the details are correct.

## What You Can Do

### Edit an existing space

Open an existing space when you need to check the setup or make a change.

- Save once the details are correct.
