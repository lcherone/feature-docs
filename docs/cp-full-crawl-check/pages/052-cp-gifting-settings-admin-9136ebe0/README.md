*Gifting Settings page overview*

## Using This Page

1. Open a Gifting Setting entry from the listing, or select Create new.
2. Complete the labelled settings for the entry.
3. Select Save to apply the changes.

## What You Can Do

### Create a new entry

Select Create new to add a Gifting Setting entry, then complete the labelled settings and save.

### Edit an existing entry

Open an existing Gifting Setting entry to review or update its settings.

- Save applies the changes.
