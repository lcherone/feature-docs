## Using This Page

1. Open a Auto Renewal Notification Setting entry from the listing, or select Create new.
2. Complete the labelled settings for the entry.
3. Select Save to apply the changes.

## What You Can Do

### Create a new entry

Select Create new to add a Auto Renewal Notification Setting entry, then complete the labelled settings and save.

### Edit an existing entry

Open an existing Auto Renewal Notification Setting entry to review or update its settings.

- Save applies the changes.

## Key Settings

The sections below highlight the settings people are most likely to change.

### Auto Renewal Notification Settings

#### Upcoming Renewal Notification Enabled

![Upcoming Renewal Notification Enabled](images/field-003-upcoming-renewal-notification-enabled.png)

*Upcoming Renewal Notification Enabled setting*

Enable or disable Upcoming Renewal Notification Enabled.

**Effect:** Updates Upcoming Renewal Notification Enabled.

#### Upcoming Renewal Notification Subject

![Upcoming Renewal Notification Subject](images/field-004-upcoming-renewal-notification-subject.png)

*Upcoming Renewal Notification Subject setting*

Enter the Upcoming Renewal Notification Subject.

**Effect:** Updates Upcoming Renewal Notification Subject.

**Validation:** Required.

#### Upcoming Renewal Notification Copy

![Upcoming Renewal Notification Copy](images/field-005-upcoming-renewal-notification-copy.png)

*Upcoming Renewal Notification Copy setting*

Enter the Upcoming Renewal Notification Copy content.

**Effect:** Updates Upcoming Renewal Notification Copy.
