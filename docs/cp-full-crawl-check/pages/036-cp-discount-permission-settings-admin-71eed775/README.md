*Discount Permission Settings page overview*

## Using This Page

1. Open the Discount Permission Settings page from the CP navigation or direct URL.
2. Review the role, maximum percentage, and maximum fixed amount columns.
3. Select Create new to add a role limit, or Edit to update an existing one.

## What You Can Do

### Review role limits

Use the listing to see which admin roles already have custom discount limits set.

- Shown: Role
- Shown: Max Discount Percentage
- Shown: Max Discount Amount

### Create a role limit

Select Create new to add a discount limit for an admin role.

### Edit a role limit

Open an existing role limit to change the percentage cap or fixed amount cap.

### Set the discount limit

Choose the role, enter the maximum percentage discount, and add a fixed amount cap if needed.

- Use 100 for an unlimited percentage allowance.
- Use 0 in Max Discount Amount for no fixed-amount cap.
